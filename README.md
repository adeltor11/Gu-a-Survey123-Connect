# Manual XLSForm — ArcGIS Survey123 Connect

Documentación técnica completa de la plantilla XLSForm Advanced (v3.25) para ArcGIS Survey123 Connect.

**→ [Ver manual en línea](https://adeltor11.github.io/Guia-Survey123-Connect/)**

---

## Contenido

Cubre cada columna, tipo de pregunta y función de lógica de la plantilla Advanced incluida en Survey123 Connect:

- Estructura del archivo (hojas `survey`, `choices`, `settings`)
- Todas las columnas de la hoja `survey` explicadas en detalle
- Tipos de preguntas, apariencias y compatibilidad Field App / Web App
- Lógica condicional: `relevant`, `constraint`, `calculation`
- Grupos, repeticiones y formularios por páginas
- Expresiones XPath y la función `pulldata()`
- HTML en labels y notas
- Flujo de trabajo completo de publicación
- Errores comunes y soluciones
- Ejemplo práctico de inventario arbóreo
- Glosario de términos

## Uso

Este repositorio contiene un sitio estático. Se puede navegar directamente en GitHub Pages o clonar para uso local.

```bash
git clone https://github.com/adeltor11/survey123-xlsform-manual.git
cd survey123-xlsform-manual
# Abrir index.html en el navegador
```

## Contexto

Desarrollado en el contexto del sistema de inventario arbóreo y estimación de carbono de la Universidad Nacional de Costa Rica (UNA), integrado con PPCN 2.0.

Tecnologías relacionadas: PostgreSQL 16, PostGIS, ArcGIS Pro, Survey123 Connect, Trimble TDC6.
